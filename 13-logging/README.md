# Logging Frameworks

## Overview
Learn application logging using Log4J and SLF4J for debugging and monitoring.

## Learning Objectives
By the end of this module, you will be able to:
- Configure Log4J 2 for application logging
- Use appropriate log levels and appenders
- Implement SLF4J as a logging facade
- Apply logging best practices

## Topics Covered

### 1. [Log4J Framework](./topics/01-log4j-framework.md)
- Log4J Configuration
- Appenders (Console, File, Rolling File)
- Layouts (Pattern Layout)
- Log4J 2 Architecture

### 2. [SLF4J (Simple Logging Facade for Java)](./topics/02-slf4j.md)
- SLF4J API, Binding with Log4J
- Logger Factory
- Best Practices for Logging
