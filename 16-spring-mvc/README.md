# Spring MVC

## Overview
Spring MVC is a web framework for building web applications using the Model-View-Controller pattern.

## Learning Objectives
By the end of this module, you will understand and be able to apply the key concepts and practices of Spring MVC.

## Topics Covered

### 1. [Spring MVC Architecture](./topics/01-spring-mvc-architecture.md)
- What is Spring MVC?
- MVC Design Pattern
- Spring MVC Architecture and Components
- Request Processing Flow
- Configuration (XML, Java, Spring Boot)
- Project Structure

### 2. [DispatcherServlet](./topics/02-dispatcher-servlet.md)
- Front Controller Pattern
- DispatcherServlet Lifecycle
- Configuration and Initialization
- Request Processing Flow
- Context Hierarchy
- Multiple DispatcherServlets

### 3. [Controllers](./topics/03-controllers.md)
- Creating Controllers
- Request Mapping (@RequestMapping, @GetMapping, etc.)
- Handler Method Parameters (@PathVariable, @RequestParam, @RequestBody)
- Return Types (String, ModelAndView, ResponseEntity)
- Form Handling and Validation
- REST Controllers

### 4. [ViewResolver](./topics/04-view-resolver.md)
- ViewResolver Purpose and Types
- InternalResourceViewResolver (JSP)
- ThymeleafViewResolver
- FreeMarkerViewResolver
- ViewResolver Chain
- Redirect and Forward

### 5. [Model and View](./topics/05-model-and-view.md)
- Working with Model Interface
- ModelAndView Object
- @ModelAttribute Annotation
- Session Attributes
- Flash Attributes
- Request and Session Scope

### 6. [Spring ORM](./topics/06-spring-orm.md)
- Spring ORM Module
- ORM Framework Integration
- DataSource Configuration
- JPA Configuration
- Entity Classes
- Repository Layer
- Transaction Management

### 7. [Hibernate Integration](./topics/07-hibernate-integration.md)
- Hibernate with Spring
- SessionFactory Configuration
- Entity Mapping
- DAO Layer with Hibernate
- HQL and Criteria API
- Batch Operations
- Service Layer with Transactions
