# RxJS (Reactive Extensions for JavaScript)

## Overview
RxJS is a library for reactive programming using observables to handle asynchronous data streams.

## Learning Objectives
By the end of this module, you will understand and be able to apply the key concepts and practices of RxJS (Reactive Extensions for JavaScript).

## Topics Covered

### Core RxJS Concepts
1. [Introduction to RxJS](./topics/01-rxjs-introduction.md) - What is RxJS, reactive programming, and why use RxJS
2. [Observables](./topics/02-observables.md) - Creating Observables, subscribing, lifecycle, hot vs cold
3. [Operators](./topics/03-operators.md) - Transformation, filtering, combination, and higher-order operators
4. [Subjects](./topics/04-subjects.md) - Subject, BehaviorSubject, ReplaySubject, and AsyncSubject

### TypeScript Type System
5. [TypeScript Generics and Utilities](./topics/05-typescript-generics.md) - Basic Generics, keyof, readonly, utility types, and array generics
