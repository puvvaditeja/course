# Introduction to CSS3

## What is CSS?

Cascading Style Sheets (CSS) is a stylesheet language used to describe the presentation of HTML documents. While HTML provides the structure and content, CSS controls the visual appearance, layout, and design of web pages.

### Key Characteristics

- **Separation of Concerns**: Keeps content (HTML) separate from presentation (CSS)
- **Cascading**: Multiple styles can apply to elements, with specific rules determining priority
- **Selective Styling**: Target specific elements using powerful selectors
- **Responsive**: Adapt layouts to different screen sizes and devices
- **Maintainable**: Change the entire site's appearance by modifying CSS files
- **Browser Support**: Widely supported across all modern browsers

### CSS3 Overview

CSS3 is the latest evolution of CSS, introducing powerful new features:

- Advanced selectors and pseudo-classes
- Transitions and animations
- Flexbox and Grid layout systems
- Media queries for responsive design
- Custom properties (CSS variables)
- Gradients, shadows, and visual effects
- Transform and filter properties

## CSS Syntax

CSS follows a simple and consistent syntax pattern:

```css
selector {
  property: value;
  property: value;
}
```

### Anatomy of a CSS Rule

```css
h1 {
  color: #333333;
  font-size: 32px;
  font-weight: bold;
  margin-bottom: 20px;
}
```

**Components:**
- **Selector** (`h1`): Identifies which HTML elements to style
- **Declaration Block** (`{}`): Contains style declarations
- **Property** (`color`, `font-size`): Aspect of the element to style
- **Value** (`#333333`, `32px`): Setting for the property
- **Declaration** (`color: #333333;`): Property-value pair ending with semicolon

### Multiple Selectors

```css
/* Apply same styles to multiple elements */
h1, h2, h3 {
  font-family: 'Arial', sans-serif;
  color: #2c3e50;
}

/* Nested rules */
.container {
  max-width: 1200px;
  margin: 0 auto;
}

.container .header {
  padding: 20px;
  background-color: #f5f5f5;
}
```

### Comments in CSS

```css
/* Single-line comment */

/*
  Multi-line comment
  Useful for longer explanations
  or temporarily disabling code
*/

.button {
  background: blue; /* Inline comment */
}
```

## Ways to Apply CSS

There are three primary methods to add CSS to HTML documents, each with specific use cases.

### 1. Inline CSS

CSS applied directly to HTML elements using the `style` attribute.

```html
<h1 style="color: blue; font-size: 24px;">Hello World</h1>
<p style="color: gray; line-height: 1.6;">This is a paragraph.</p>
```

**Pros:**
- Highest specificity (overrides other styles)
- Quick for testing or single-use styles

**Cons:**
- Not reusable
- Mixes content with presentation
- Difficult to maintain
- Cannot use pseudo-classes or media queries

**When to Use:** Rarely. Only for email templates or dynamic styles generated by JavaScript.

### 2. Internal CSS (Embedded)

CSS placed in the `<style>` tag within the HTML document's `<head>`.

```html
<!DOCTYPE html>
<html>
<head>
  <title>My Page</title>
  <style>
    body {
      font-family: Arial, sans-serif;
      margin: 0;
      padding: 0;
    }

    h1 {
      color: #2c3e50;
      text-align: center;
    }

    .container {
      max-width: 1200px;
      margin: 0 auto;
      padding: 20px;
    }
  </style>
</head>
<body>
  <div class="container">
    <h1>Welcome</h1>
  </div>
</body>
</html>
```

**Pros:**
- Keeps styles with the HTML file
- Can use all CSS features
- Good for single-page applications

**Cons:**
- Not reusable across multiple pages
- Increases HTML file size
- Makes caching less efficient

**When to Use:** Single-page websites, email templates, or component-specific styles.

### 3. External CSS (Recommended)

CSS written in separate `.css` files and linked to HTML documents.

**styles.css:**
```css
body {
  font-family: 'Segoe UI', Tahoma, Geneva, Verdana, sans-serif;
  line-height: 1.6;
  color: #333;
  margin: 0;
  padding: 0;
}

.header {
  background-color: #2c3e50;
  color: white;
  padding: 20px 0;
  text-align: center;
}

.container {
  max-width: 1200px;
  margin: 0 auto;
  padding: 20px;
}
```

**index.html:**
```html
<!DOCTYPE html>
<html>
<head>
  <title>My Website</title>
  <link rel="stylesheet" href="styles.css">
</head>
<body>
  <header class="header">
    <h1>My Website</h1>
  </header>
  <div class="container">
    <p>Content goes here</p>
  </div>
</body>
</html>
```

**Pros:**
- Reusable across multiple pages
- Separation of concerns
- Cacheable by browsers
- Easier to maintain
- Cleaner HTML

**Cons:**
- Requires additional HTTP request
- Slightly more setup

**When to Use:** All multi-page websites and production applications.

### Multiple CSS Files

```html
<head>
  <!-- General styles -->
  <link rel="stylesheet" href="css/reset.css">
  <link rel="stylesheet" href="css/main.css">

  <!-- Component styles -->
  <link rel="stylesheet" href="css/header.css">
  <link rel="stylesheet" href="css/footer.css">

  <!-- Page-specific styles -->
  <link rel="stylesheet" href="css/home.css">
</head>
```

**Loading Order Matters:** Styles are applied in order, with later styles overriding earlier ones.

## CSS Selectors

Selectors are patterns used to target HTML elements for styling. CSS3 provides powerful and flexible selector options.

### Basic Selectors

#### Universal Selector

Selects all elements.

```css
* {
  margin: 0;
  padding: 0;
  box-sizing: border-box;
}
```

#### Type Selector (Element Selector)

Selects all elements of a specific type.

```css
p {
  line-height: 1.6;
}

h1 {
  font-size: 2.5em;
}

a {
  text-decoration: none;
}
```

#### Class Selector

Selects elements with a specific class attribute. Prefix with `.`

```css
.button {
  padding: 10px 20px;
  background-color: #3498db;
  color: white;
  border: none;
}

.highlight {
  background-color: yellow;
}

.container {
  max-width: 1200px;
  margin: 0 auto;
}
```

**HTML:**
```html
<button class="button">Click Me</button>
<p class="highlight">Important text</p>
<div class="container">Content</div>
```

#### ID Selector

Selects a single element with a specific id attribute. Prefix with `#`

```css
#header {
  background-color: #2c3e50;
  height: 80px;
}

#main-content {
  min-height: 500px;
}
```

**HTML:**
```html
<header id="header">Header</header>
<main id="main-content">Content</main>
```

**Note:** IDs should be unique on a page. Use classes for reusable styles.

### Combinator Selectors

#### Descendant Selector (Space)

Selects all descendants (nested elements at any level).

```css
.container p {
  color: #333;
}

nav a {
  color: white;
}
```

#### Child Selector (`>`)

Selects direct children only.

```css
.parent > .child {
  margin-left: 20px;
}

ul > li {
  list-style: disc;
}
```

#### Adjacent Sibling Selector (`+`)

Selects the element immediately following another.

```css
h1 + p {
  font-size: 1.2em;
  font-weight: bold;
}
```

#### General Sibling Selector (`~`)

Selects all siblings following an element.

```css
h1 ~ p {
  color: gray;
}
```

### Attribute Selectors

Select elements based on attributes.

```css
/* Has attribute */
input[type] {
  border: 1px solid #ccc;
}

/* Exact match */
input[type="text"] {
  padding: 5px;
}

/* Contains word */
a[title~="example"] {
  color: green;
}

/* Starts with */
a[href^="https"] {
  color: blue;
}

/* Ends with */
a[href$=".pdf"] {
  color: red;
}

/* Contains substring */
a[href*="example"] {
  font-weight: bold;
}
```

### Pseudo-class Selectors

Select elements based on state or position.

```css
/* Link states */
a:link { color: blue; }
a:visited { color: purple; }
a:hover { color: red; }
a:active { color: orange; }

/* First and last child */
li:first-child { font-weight: bold; }
li:last-child { border-bottom: none; }

/* Nth child */
tr:nth-child(odd) { background-color: #f5f5f5; }
tr:nth-child(even) { background-color: white; }
tr:nth-child(3) { color: red; }

/* Form states */
input:focus { border-color: blue; }
input:disabled { opacity: 0.5; }
input:checked { background-color: green; }

/* Not selector */
p:not(.special) { color: gray; }
```

### Pseudo-element Selectors

Style specific parts of elements.

```css
/* First letter */
p::first-letter {
  font-size: 2em;
  font-weight: bold;
}

/* First line */
p::first-line {
  font-variant: small-caps;
}

/* Before and after */
.quote::before {
  content: '"';
  font-size: 2em;
}

.quote::after {
  content: '"';
  font-size: 2em;
}

/* Selection */
::selection {
  background-color: yellow;
  color: black;
}
```

### Selector Specificity

When multiple rules target the same element, specificity determines which wins.

**Specificity Hierarchy (lowest to highest):**
1. Universal selector (`*`)
2. Type selectors (`p`, `div`)
3. Class selectors (`.class`)
4. ID selectors (`#id`)
5. Inline styles (`style=""`)
6. `!important` (use sparingly!)

**Calculation:**
- ID = 100 points
- Class/Attribute/Pseudo-class = 10 points
- Element/Pseudo-element = 1 point

```css
p { color: black; }              /* Specificity: 1 */
.text { color: blue; }            /* Specificity: 10 */
p.text { color: green; }          /* Specificity: 11 */
#content .text { color: red; }    /* Specificity: 110 */
```

## Styling Text and Fonts

### Font Properties

#### Font Family

```css
body {
  font-family: 'Helvetica Neue', Helvetica, Arial, sans-serif;
}

/* Web-safe fonts */
.serif { font-family: Georgia, 'Times New Roman', serif; }
.sans-serif { font-family: Arial, Verdana, sans-serif; }
.monospace { font-family: 'Courier New', Courier, monospace; }

/* Google Fonts */
@import url('https://fonts.googleapis.com/css2?family=Roboto:wght@300;400;700&display=swap');

.custom-font {
  font-family: 'Roboto', sans-serif;
}
```

#### Font Size

```css
/* Absolute units */
p { font-size: 16px; }

/* Relative units (recommended) */
h1 { font-size: 2em; }      /* 2x parent font size */
h2 { font-size: 1.5rem; }   /* 1.5x root font size */
p { font-size: 1rem; }      /* Root font size */

/* Percentage */
small { font-size: 80%; }

/* Viewport units */
.hero { font-size: 5vw; }   /* 5% of viewport width */
```

#### Font Weight

```css
p { font-weight: normal; }   /* 400 */
strong { font-weight: bold; } /* 700 */

.thin { font-weight: 100; }
.light { font-weight: 300; }
.regular { font-weight: 400; }
.medium { font-weight: 500; }
.semibold { font-weight: 600; }
.bold { font-weight: 700; }
.extrabold { font-weight: 800; }
.black { font-weight: 900; }
```

#### Font Style

```css
em { font-style: italic; }
.normal { font-style: normal; }
.oblique { font-style: oblique; }
```

### Text Properties

#### Text Color

```css
p { color: #333333; }
h1 { color: rgb(44, 62, 80); }
.highlight { color: rgba(255, 0, 0, 0.7); }
.modern { color: hsl(200, 50%, 50%); }
```

#### Text Alignment

```css
.left { text-align: left; }
.center { text-align: center; }
.right { text-align: right; }
.justify { text-align: justify; }
```

#### Text Decoration

```css
a { text-decoration: none; }
.underline { text-decoration: underline; }
.line-through { text-decoration: line-through; }
.overline { text-decoration: overline; }

/* Multiple decorations */
.fancy {
  text-decoration: underline wavy red;
}
```

#### Text Transform

```css
.uppercase { text-transform: uppercase; }
.lowercase { text-transform: lowercase; }
.capitalize { text-transform: capitalize; }
.none { text-transform: none; }
```

#### Line Height

```css
p {
  line-height: 1.6;  /* Recommended for readability */
}

.tight { line-height: 1.2; }
.loose { line-height: 2; }
.pixel { line-height: 24px; }
```

#### Letter Spacing

```css
.spaced { letter-spacing: 2px; }
.tight { letter-spacing: -1px; }
h1 { letter-spacing: 0.05em; }
```

#### Word Spacing

```css
p { word-spacing: 5px; }
.wide { word-spacing: 10px; }
```

#### Text Shadow

```css
h1 {
  text-shadow: 2px 2px 4px rgba(0, 0, 0, 0.3);
}

.multiple-shadows {
  text-shadow:
    1px 1px 2px black,
    0 0 25px blue,
    0 0 5px darkblue;
}

/* Offset-x | Offset-y | Blur-radius | Color */
.glow {
  text-shadow: 0 0 10px #fff, 0 0 20px #fff, 0 0 30px #e60073;
}
```

### Complete Text Styling Example

```css
.article {
  font-family: 'Georgia', serif;
  font-size: 18px;
  line-height: 1.8;
  color: #333;
  text-align: justify;
}

.article h1 {
  font-family: 'Arial', sans-serif;
  font-size: 2.5em;
  font-weight: bold;
  color: #2c3e50;
  text-transform: uppercase;
  letter-spacing: 2px;
  margin-bottom: 20px;
}

.article p {
  margin-bottom: 15px;
}

.article a {
  color: #3498db;
  text-decoration: none;
  border-bottom: 1px solid #3498db;
}

.article a:hover {
  color: #2980b9;
  border-bottom-color: #2980b9;
}
```

## Best Practices

### 1. Use External Stylesheets

Keep CSS in separate files for maintainability and reusability.

### 2. Follow Naming Conventions

```css
/* Use descriptive class names */
.navigation-menu { }        /* Good */
.nm { }                     /* Bad */

/* Use BEM (Block Element Modifier) methodology */
.button { }                 /* Block */
.button__icon { }           /* Element */
.button--primary { }        /* Modifier */
```

### 3. Organize Your CSS

```css
/* 1. Reset/Normalize */
* { margin: 0; padding: 0; }

/* 2. Variables and imports */
@import url('fonts.css');

/* 3. Base styles */
body { }
h1, h2, h3 { }
p { }

/* 4. Layout */
.container { }
.header { }
.footer { }

/* 5. Components */
.button { }
.card { }

/* 6. Utilities */
.text-center { }
.mb-20 { }
```

### 4. Avoid Over-Specificity

```css
/* Bad - Too specific */
div#content .wrapper ul li a { }

/* Good - Just specific enough */
.nav-link { }
```

### 5. Use Comments

```css
/* ==========================================================================
   Header Component
   ========================================================================== */

.header {
  /* Styles */
}

/* Navigation
   ========================================================================== */

.nav {
  /* Styles */
}
```

### 6. Mobile-First Approach

Start with mobile styles, then add desktop styles with media queries.

### 7. Use Shorthand Properties

```css
/* Instead of */
margin-top: 10px;
margin-right: 20px;
margin-bottom: 10px;
margin-left: 20px;

/* Use */
margin: 10px 20px;
```

### 8. Validate Your CSS

Use tools like W3C CSS Validator to catch errors.

| CSS Syntax | `selector { property: value; }` |

| Combinators | Descendant (` `), Child (`>`), Sibling (`+`, `~`) |
