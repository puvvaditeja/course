# JavaScript

## Overview
JavaScript is a dynamic programming language for web development. This module covers ES6+ features, DOM manipulation, asynchronous programming, and modern JavaScript concepts.

## Learning Objectives
By the end of this module, you will understand and be able to apply the key concepts and practices of JavaScript.

## Topics Covered

### 1. [JavaScript Fundamentals](./topics/01-javascript-fundamentals.md)
- What is JavaScript?
- Getting Started with JavaScript
- Variables (var, let, const)
- Data Types (primitives and reference types)
- Operators (arithmetic, comparison, logical, ternary)
- Control Flow (if-else, switch, loops)
- Best practices

### 2. [Functions and Scope](./topics/02-functions-and-scope.md)
- Function Declaration and Expression
- Arrow Functions
- Callbacks and Higher-Order Functions
- Closures
- IIFE (Immediately Invoked Function Expression)
- Scope (global, function, block, lexical)
- The 'this' keyword

### 3. [DOM Manipulation](./topics/03-dom-manipulation.md)
- Document Object Model (DOM)
- Selecting Elements (getElementById, querySelector)
- Modifying Content (textContent, innerHTML)
- Traversing the DOM (parent, children, siblings)
- Creating and Removing Elements
- Events and Event Listeners
- Event Propagation and Delegation

### 4. [Objects and JSON](./topics/04-objects-and-json.md)
- JavaScript Objects
- Object Creation (literal, constructor, class)
- Properties and Methods
- The 'this' keyword in objects
- Object manipulation (assign, freeze, seal)
- JSON (JavaScript Object Notation)
- JSON.parse() and JSON.stringify()
- Working with APIs

### 5. [Asynchronous JavaScript](./topics/05-asynchronous-javascript.md)
- Synchronous vs Asynchronous
- Call Stack and Execution Context
- Event Loop and Callback Queue
- Callbacks and Callback Hell
- Promises (then, catch, finally)
- async/await
- Fetch API
- Error handling

### 6. [ES6+ Features](./topics/06-es6-features.md)
- let and const
- Arrow Functions
- Template Literals
- Destructuring (arrays and objects)
- Spread and Rest Operators
- Enhanced Object Literals
- Modules (import/export)
- Classes and Inheritance
- Default Parameters
